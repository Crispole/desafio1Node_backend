Desafío - Veterinaria Js

En este desafío pondremos a prueba nuestros primeros conocimientos de Node Js desarrollando una pequeña aplicación backend que registre 
las horas de atención en una veterinaria.

Para ejercutar el desafio debes:

1.- Abrir la terminal

2.- Para registrar debes: ejecutar node + index.js + registrar y añadir los 5 argumentos necesarios para ingresar la cita. Ejemplo: node index.js registrar Cuchufli "3 años" gato naranjo "perdida de orientacion".

3.- Para leer la cita debes: ejecutar node + index.js + leer y aparecerán todas las citas registradas.

4.- Para vaciar las citas debe: ejecutar node + index.js + vaciar y vaciará todo lo creado
